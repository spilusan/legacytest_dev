Listing on Pages

All ShipServ Members receive a Premium Profile, the benefits of a Premium Profile include:

  • Greater brand awareness, premium profiles show much more information than a basic listing, company description, logo, ability to attach documents & online catalogues.
  • Appear in much more search results, you can select all supplier categories within a premium profile rather than 3 in a basic profile. As a result you appear in more search results.
  • All premium profiles are displayed above ALL basic listings in Pages, as a results premium profiles get more contact views, company profile views & more RFQs through Pages.

On average, we see Premium Profile suppliers getting 20x more RFQs through Pages.
